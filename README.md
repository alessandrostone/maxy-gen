# Maxy::Gen

A commandline tool to generate max patches in an emmet-like fashion

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'maxy-gen'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install maxy-gen
